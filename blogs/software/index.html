Software

Our primary software tools are developed as modular R packages, including:

  1. osmdata, a package for very efficient and flexible extraction of data from Open Street Map (OSM).
  2. dodgr (Distances on Directed Graphs in R), a package for highly efficient and scalable routing and allocation of aggregate flows on complex, dual-weighted graphs.
